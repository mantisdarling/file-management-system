# Project Notes

I built this small File Management System using Python while learning file handling.

---

## Why I made this

I wanted to understand how Python actually works with real files on my computer, not just theory. So I tried building something practical where I can create, read, update, and delete files.

---

## What this project does

It’s a simple command-line program where you can:

- Create files  
- Read files  
- Update files (rename, add text, replace content)  
- Delete files  
- See all files in a folder  

Basically, it acts like a mini file manager.

---

## Problems I faced as a beginner

While making this, I got stuck a lot.

- I didn’t fully understand how file paths work  
- I kept getting errors like EOFError when running input()  
- Small mistakes like wrong indentation or missing .strip() broke the code  
- I also didn’t know how to properly structure code into functions at first  
- Debugging took more time than writing the actual code  

Honestly, most of the time I was just fixing errors and learning from them.

---

## What I learned

Even though it’s a small project, I learned a lot:

- How Python handles files in real life  
- How input/output works in terminal programs  
- How to organize code using functions  
- How to debug step by step instead of panicking  
- How real programs are built slowly, not in one go  

---

## Why it matters to me

It’s not a big project, but it helped me understand how coding actually feels in real life. I made mistakes, fixed them, and learned a lot from it.

This is just the beginning.
Not perfect, just progressing.
