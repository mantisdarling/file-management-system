# File Management System

A Python-based file management system that allows users to:

- Create files
- Read files
- Update files
- Delete files
- List files in the current directory

## Technologies Used

- Python
- pathlib

## How to Run

python file_management_system.py


## Requirements

* Python 3.8 or newer
* Run from a terminal that supports interactive input

Example:

python file_manager.py

Some online code runners may not support input() and can produce EOFError.
